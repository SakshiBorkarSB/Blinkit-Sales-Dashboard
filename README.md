# Blinkit-Sales-Dashboard
This project presents a comprehensive Sales Analysis Dashboard for Blinkit – India's Last Minute App, built using Power BI.
The objective of this project is to analyze sales performance, outlet distribution, product categories, ratings, and customer behavior to generate actionable business insights.
The dashboard provides interactive filtering and KPI tracking to help stakeholders understand trends and performance across different outlet types and locations.

# Dataset Overview:
Total Records – 8523 
Total Fields – 12 

# Business Objectives
- Analyze total sales performance across outlets
- Identify high-performing item categories
- Compare outlet sizes and location tiers
- Evaluate customer ratings and item visibility
- Track sales trends over establishment years
- Understand impact of fat content on sales

# Key KPIs
- Total Sales: The overall revenue generated from all the items sold.
- Average Sales: The average revenue per sale.
- Number of items: The total count of different items sold.
- Average rating: The average customer rating for items sold. 

# Dashboard Features
1. Sales Trend Analysis
Outlet Establishment Year vs Sales
Year-wise growth pattern
Peak sales identification

2. Product Category Insights
Sales by Item Type
Top performing categories (Fruits, Snacks, Household etc.)
Fat Content Analysis (Low Fat vs Regular)

3. Outlet Analysis
Sales by Outlet Size (Small, Medium, High)
Sales by Outlet Location Tier (Tier 1, Tier 2, Tier 3)
Outlet Type comparison:
Supermarket Type1
Supermarket Type2
Supermarket Type3
Grocery Store

4. Interactive Filter Panel
Users can filter data by:
Outlet Location Type
Outlet Size
Item Type
Dashboard dynamically updates based on selected filters.

# Tools & Technologies Used
- Power BI
- Data Cleaning & Transformation (Power Query)
- DAX for KPI calculations
- Interactive Visualizations
- Dataset: Blinkit Sales Dataset (included in repository)

# Insights Derived
- Supermarket Type1 generates the highest total sales.
- Tier 3 locations contribute the highest revenue.
- Medium-sized outlets outperform small and high-sized outlets.
- Fruits & Snacks categories contribute significantly to total revenue.
- Majority of products maintain an average rating of 3.9.
